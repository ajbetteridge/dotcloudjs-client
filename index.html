dotcloud.js

dotcloud.js is a Javascript library which lets you interact with powerful backend services directly from the browser, in just a few lines of code. For example, here's how to store data on a database:

dotcloud.db.insert('todo', {
      description: 'Learn how to play foosball',
      priority: 'high'
});

Here's how to retrieve data back from the database:

dotcloud.db.find('todo',
{priority: 'high'}, function (data) { /* ... */ });
